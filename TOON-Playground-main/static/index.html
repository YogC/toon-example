What is TOON?

TOON is a compact, token-efficient data format for LLMs. It combines YAML-like indentation with CSV-style tabular arrays, reducing token usage by 30-60% compared to JSON while maintaining full compatibility.

Declare Fields Once

                    
Instead of repeating field names for each object, TOON declares them once in a header.

                    

                        

                            JSON (Repetitive)
                            
{"id": 1, "name": "Alice"}
{"id": 2, "name": "Bob"}

                            TOON (Efficient)

  1,Alice
